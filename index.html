Connecting to API

The focus of this project was to create a script that would automatically pull data from Coinmarketcap's API.
The user gets to choose how many times the data will be pulled and how long the time interval is between each pull. So if you wanted to gather the price of Bitcoin for the whole week, you can specify that you want to pull the data 7 times, with a time interval of 1440 minutes between each pull. After the data gathering is complete the script will then clean the data by removing unwanted characters and text making it ready for use.

  • Jupyter Notebook

Analyzing the Movie Industry

The data for this project was downloaded from Kaggle. Purpose of this project is to demonstrate my thought process of how I go about cleaning and manipulating the data when certain problems come up. One of the problems I faced with this dataset was that there was about 30% of the data missing within the budget column.
I had to try a couple of methods to deal with the missing values:

  • Removing the Null values and then just analyze the remaining data
    • or
  • Creating a predictive model that will fill in the missing data
Both these solutions have their own negative apects.

Project is split in 4 sections:
  1. Data Cleaning
  2. Creation of a Linear Regression Model
  3. Visualization
  4. Creating a Csv/Excel File.
